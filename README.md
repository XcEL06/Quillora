- **PWA Features:** manifest.json + service-worker.js for offline support
- **Offline Caching:** Service Worker caches pages and gallery assets
- **Local AI Writing Assist:** Predefined suggestions for writing prompts
- **Hosting:** Compatible with GitHub Pages, Netlify, Vercel
